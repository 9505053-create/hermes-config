# 對話視窗管理

## 觸發指令
- `~~` → 壓縮當前對話重點，存檔，請使用者打 `/new`
- `@@` → 存檔，刪除摘要，請使用者打 `/new`

## 共同步驟（兩者都會執行）

### 存檔到硬碟
將當前對話完整摘要存入 `~/.hermes/memory/archive_{YYYYMMDD_HHMMSS}.md`
作為日後調閱參考，不主動刪除。

## 指令：~~（壓縮並帶入）

### 執行步驟
1. 存檔（共同步驟）
2. 梳理當前對話的關鍵決策、進度、待辦事項
3. 寫入 `~/.hermes/memory/last_session_summary.md`
4. 回覆：「✅ 已存檔。輸入 `/new` 開新視窗，我會自動帶入摘要。」

### 摘要模板
```markdown
# 會話摘要 — {日期時間}

## 本輪決策
- {決策1}

## 完成事項
- {事項1}

## 待辦/未完成
- {待辦1}

## 重要上下文
- {背景資訊}

## 技能異動
- {本次新增/修改的技能}
```

## 指令：@@（純開新視窗）

### 執行步驟
1. 存檔（共同步驟）
2. 刪除 `~/.hermes/memory/last_session_summary.md`（如有）
3. 回覆：「✅ 已存檔。輸入 `/new` 開新視窗，全新對話。」

## 新對話啟動時（自動行為）

1. 檢查 `~/.hermes/memory/last_session_summary.md` 是否存在
2. 若存在 → 讀取，回報摘要內容，刪除檔案
3. 若不存在 → 正常啟動

## 自動監控

- 對話超過 **40 輪**或工具呼叫超過 **20 次**時，提醒 Scott 考慮 `~~`
- 僅提醒一次
